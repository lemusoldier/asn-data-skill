# 01 — Python Dasar untuk ASN

> Pengenalan Python untuk pegawai negeri yang belum pernah coding.
> Semua script di folder ini bisa dijalankan tanpa install library tambahan.

## Cara Menjalankan

```bash
# Buka terminal / Command Prompt
python 01_hello.py
python 02_variabel.py
python 03_kondisi.py
python 04_perulangan.py
python 05_list_dict.py
```

## Topik

| File | Isi |
|------|-----|
| `01_hello.py` | Hello world & cetak data |
| `02_variabel.py` | Tipe data & variabel |
| `03_kondisi.py` | If/else (misal: golongan → tunjangan) |
| `04_perulangan.py` | Loop (misal: cetak daftar nama) |
| `05_list_dict.py` | List & Dictionary (struktur data ASN) |
