## 📖 Daftar Lengkap Fungsi Excel untuk ASN

### Fungsi Dasar (Level 1)

| Fungsi | Kegunaan | Contoh Kasus ASN |
|--------|----------|-----------------|
| `SUM` | Menjumlahkan angka | Total gaji pokok seluruh pegawai |
| `AVERAGE` | Rata-rata | Rata-rata masa kerja |
| `COUNT` | Hitung angka | Jumlah pegawai aktif |
| `COUNTA` | Hitung non-kosong | Jumlah pegawai yang sudah input data |
| `MAX` / `MIN` | Nilai tertinggi/terendah | Gaji tertinggi & terendah |
| `MEDIAN` | Nilai tengah | Gaji tengah (lebih stabil dari rata-rata) |
| `SUMIF` | Jumlah dengan 1 syarat | Total gaji pegawai golongan IV |
| `SUMIFS` | Jumlah dengan banyak syarat | Total gaji gol IV di unit Kepegawaian |
| `COUNTIF` | Hitung dengan 1 syarat | Berapa pegawai yang sudah pensiun |
| `COUNTIFS` | Hitung dengan banyak syarat | Berapa pegawai gol III di unit Keuangan |
| `AVERAGEIF` | Rata-rata dengan syarat | Rata-rata gaji pegawai gol III |

### Fungsi Logika (Level 1)

| Fungsi | Kegunaan | Contoh Kasus ASN |
|--------|----------|-----------------|
| `IF` | Kondisi sederhana | Jika gol IV → tunjangan 1.5jt, selain itu → 1jt |
| `IFS` | Banyak kondisi sekaligus | Gol I→500k, II→750k, III→1jt, IV→1.5jt |
| `AND` | Semua syarat terpenuhi | Cek apakah usia ≥ 30 DAN masa kerja ≥ 5 tahun |
| `OR` | Salah satu syarat terpenuhi | Cek apakah gol IV ATAU masa kerja ≥ 20 tahun |
| `NOT` | Membalik kondisi | Tampilkan pegawai yang BUKAN gol IV |

### Fungsi Teks (Level 2) — Penting untuk Data Nama & NIP

| Fungsi | Kegunaan | Contoh Kasus ASN |
|--------|----------|-----------------|
| `CONCAT` | Gabung teks tanpa pemisah | Gabung kode unit + nomor urut → "KEP001" |
| `CONCATENATE` | Gabung teks (versi lama) | Sama seperti CONCAT |
| `TEXTJOIN` | Gabung teks dengan delimiter | Gabung nama depan + belakang pakai spasi |
| `TRIM` | Hapus spasi berlebih | Rapikan nama dari input yang berantakan |
| `CLEAN` | Hapus karakter tersembunyi | Bersihkan data dari copy-paste sistem |
| `PROPER` | Huruf kapital awal kata | "budi santoso" → "Budi Santoso" |
| `UPPER` / `LOWER` | Huruf besar/kecil semua | Standarisasi nama jabatan |
| `SUBSTITUTE` | Ganti teks tertentu | Ganti "III/c" jadi "Penata Muda" untuk laporan |
| `LEFT` | Ambil karakter dari kiri | Ambil 2 digit pertama NIP → kode provinsi |
| `RIGHT` | Ambil karakter dari kanan | Ambil 4 digit terakhir NIP → nomor urut |
| `MID` | Ambil karakter dari tengah | Ambil tanggal lahir dari NIP (digit 7-8) |
| `LEN` | Hitung jumlah karakter | Cek apakah NIP tepat 18 digit |
| `FIND` / `SEARCH` | Cari posisi teks | Cek apakah "@" ada di alamat email |
| `VALUE` | Teks → angka | Konversi "5000000" jadi angka untuk dijumlah |
| `TEXT` | Angka → teks dengan format | Format tanggal: 44562 → "15 Juni 2022" |
| `REPT` | Ulang teks | Bikin visualisasi sederhana: "███" untuk rating |

### Fungsi Lookup (Level 2) — Mencari Data Antar Tabel

| Fungsi | Kegunaan | Contoh Kasus ASN |
|--------|----------|-----------------|
| `VLOOKUP` | Cari data secara vertikal | Cari nama pegawai berdasarkan NIP |
| `HLOOKUP` | Cari data secara horizontal | Cari data dari tabel format baris |
| `XLOOKUP` | Cari data (versi modern, fleksibel) | Pengganti VLOOKUP yang lebih powerful |
| `INDEX` | Ambil nilai dari posisi tertentu | Ambil gaji dari tabel berdasarkan baris & kolom |
| `MATCH` | Cari posisi data | Cari posisi golongan "IV/a" di daftar |
| `INDEX + MATCH` | Kombinasi powerful | Cari data tanpa batasan arah kolom |
| `CHOOSE` | Pilih nilai dari indeks | Pilih tunjangan berdasarkan level (1/2/3) |
| `SWITCH` | Cocokkan beberapa nilai | Konversi kode golongan ke nama jabatan |

### Fungsi Tanggal & Waktu (Level 2)

| Fungsi | Kegunaan | Contoh Kasus ASN |
|--------|----------|-----------------|
| `TODAY` | Tanggal hari ini | Hitung usia = TAHUN(TODAY()) - TAHUN_lahir |
| `NOW` | Tanggal + jam sekarang | Timestamp cetak laporan |
| `YEAR` / `MONTH` / `DAY` | Ekstrak komponen tanggal | Ambil tahun masuk dari tanggal SK |
| `WEEKDAY` | Hari dalam minggu | Cek apakah hari kerja (senin-jumat) |
| `DATEDIF` | Selisih dua tanggal | Hitung masa kerja: bulan/tahun/hari |
| `EOMONTH` | Akhir bulan | Hitung gaji perakhir bulan |
| `EDATE` | Tambah/subtract bulan | Hitung kontrak berakhir 3 bulan dari sekarang |
| `NETWORKDAYS` | Hitung hari kerja | Hitung hari kerja efektif (exclude weekend & libur) |

### Fungsi Angka & Matriks (Level 3)

| Fungsi | Kegunaan | Contoh Kasus ASN |
|--------|----------|-----------------|
| `SUMPRODUCT` | Perkalian matriks + jumlah | Hitung total anggaran: (qty × harga) per unit |
| `ROUND` / `ROUNDUP` / `ROUNDDOWN` | Pembulatan | Bulatkan gaji ke ribuan terdekat |
| `INT` | Ambil bagian bulat | Konversi desimal ke integer |
| `MOD` | Sisa bagi | Cek ganjil/genap nomor urut pegawai |
| `ABS` | Nilai absolut | Selisih anggaran (tanpa minus) |
| `POWER` / `SQRT` | Pangkat & akar | Hitung proyeksi pertumbuhan |
| `SUBTOTAL` | Total yang mengabaikan filter | Jumlah pegawai yang sedang di-filter |
