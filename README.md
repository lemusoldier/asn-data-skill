# 📊 ASN Data Skill — Silabus & Roadmap Excel Terkurasi

> Repositori terkurasi berisi roadmap pembelajaran Excel untuk Aparatur Sipil Negara (ASN), admin, dan staf perkantoran. Semua materi dilengkapi dengan link video YouTube & TikTok yang sudah diverifikasi.

---

## 🗺️ Roadmap Pembelajaran (Silabus)

Untuk menguasai Excel demi keperluan administrasi dan pelaporan, silakan ikuti alur belajar dari Level 1 hingga Level 4. Setiap materi didesain sesuai dengan *case study* atau kebutuhan riil di instansi pemerintahan.
