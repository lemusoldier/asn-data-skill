## 🎯 Kenapa Excel Penting untuk ASN?

Sebagian besar pekerjaan ASN melibatkan pengolahan data: rekapitulasi, pelaporan, penggajian, kepegawaian, hingga analisis anggaran. Menguasai Excel artinya kerja lebih cepat, lebih akurat, dan lebih efisien.

## 📋 Daftar Skill Excel untuk ASN

### 1. 🧹 Data Cleaning & Organization

> Skill paling dasar tapi paling sering dibutuhkan. Data dari sistem (SAPK, SIM, e-Office) jarang yang langsung rapi.

**Yang dipelajari:**
- **Filter & Sort** — Menyaring dan mengurutkan data
- **Remove Duplicates** — Menghapus data duplikat
- **Text to Columns** — Memisahkan data dalam satu kolom
- **Flash Fill** — Auto-fill pola data secara otomatis
- **Find & Replace** — Mencari dan mengganti data massal

### 3. 🔍 Lookup & Reference

> Skill untuk mencari dan mengambil data dari tabel lain — wajib dikuasai untuk rekapitulasi data pegawaian, NIP, golongan, dll.

**Yang dipelajari:**
- **VLOOKUP** — Mencari data secara vertikal
- **HLOOKUP** — Mencari data secara horizontal
- **XLOOKUP** — Versi modern yang lebih fleksibel
- **INDEX + MATCH** — Kombinasi powerful untuk pencarian data
- **CHOOSE** — Memilih nilai berdasarkan indeks

### 5. 📅 Date & Time Functions

> Penting untuk menghitung masa kerja, masa jabatan, deadline, dan pelaporan periode.

**Yang dipelajari:**
- **TODAY, NOW** — Tanggal dan waktu saat ini
- **YEAR, MONTH, DAY** — Mengekstrak komponen tanggal
- **DATEDIF** — Menghitung selisih waktu (masa kerja)
- **EOMONTH** — Akhir bulan
- **NETWORKDAYS** — Menghitung hari kerja
- **EDATE** — Menambah/subtract bulan
