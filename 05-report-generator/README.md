# 05 — Report Generator

> Generate laporan ringkasan otomatis dari data mentah.
> Cocok untuk laporan bulanan, rekap pegawai, analisis anggaran, dll.

## Yang Dibutuhkan

```bash
pip install pandas openpyxl
```

## Topik

| File | Fungsi |
|------|--------|
| `01_rekap_pegawai.py` | Rekap jumlah pegawai per unit/golongan |
| `02_analisis_anggaran.py` | Analisis gaji & tunjangan per unit kerja |
| `03_usia_pensiun.py` | Prediksi pensiun & data demografi usia |
| `04_ringkasan_unit.py` | Generate laporan ringkasan per-unit kerja |
