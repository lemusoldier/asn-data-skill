# 04 — Validasi Data dengan Python

> Mengecek & memvalidasi data pegawai secara otomatis.
> Cocok untuk daily check data kepegawaian, cek inkonsistensi, dll.

## Yang Dibutuhkan

```bash
pip install pandas
```

## Topik

| File | Fungsi |
|------|--------|
| `01_cek_nip_valid.py` | Validasi format NIP (18 digit) |
| `02_cek_range_angka.py` | Cek angka di luar batas wajar (gaji, usia) |
| `03_cek_inkonsistensi.py` | Cari data tidak konsisten (nama sama, NIP beda) |
| `04_laporan_otomatis.py` | Generate ringkasan validasi data (summary report) |
