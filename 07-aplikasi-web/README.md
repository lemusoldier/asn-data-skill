# 07 — Aplikasi Web (Streamlit) Dashboard ASN

> Membuat interface visual sederhana (Web GUI) agar rekan kerja ASN yang tidak tahu coding bisa menggunakan script pembersih data Anda dengan klik-klik di browser.

## Yang Dibutuhkan

```bash
pip install streamlit pandas openpyxl
```

## Cara Menjalankan

```bash
streamlit run app.py
```

## Fitur Aplikasi

1.  **Pembersih Data Duplikat** — Upload Excel/CSV kotor, klik tombol, download data bersih.
2.  **Visualisasi Cepat** — Melihat grafik sebaran golongan dan unit kerja secara instan.
3.  **Merge Data** — Upload beberapa file, otomatis gabung jadi satu file Excel.
